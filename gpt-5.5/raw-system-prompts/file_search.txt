## Namespace: file_search

### Target channel: analysis

### Description
Tool for searching and viewing files uploaded directly in this conversation. Use the tool when the uploaded-file context already in the conversation is not sufficient.

To invoke, send a message in the `analysis` channel with the recipient set as `to=file_search.<function_name>`.
- To call `file_search.msearch`, use: `file_search.msearch({"queries": ["first query", "second query"]})`
- To call `file_search.mclick`, use: `file_search.mclick({"pointers": ["1:2", "1:4"]})`

### Effective Tool Use
- Use `msearch` to search across uploaded files only.
- Use `mclick` only to expand uploaded-file search results that were already returned by `msearch`.

### Citing Search Results
All answers must either include citations such as: `ÓąÄfileciteÓąāturn7file4ÓąāL10-L20ÓąĀ`, or file navlists such as `ÓąÄfilenavlistÓąā4:0ÓąÖ<description of 4:0>Óąā4:2ÓąÖ<description of 4:2>ÓąĀ`.

An example citation for a single line: `ÓąÄfileciteÓąāturn7file4ÓąāL5-L5ÓąĀ`

### Navlists
If the user asks to find / look for / search for / show uploaded files, use a file navlist in your response.
